# Insert Recipes

Code recipes for splicing pages from one PPTX into another. Used by `scripts/pptx_insert.py`.

**Prerequisites:** Read `references/insert/workflow.md` first to understand the overall flow.

---

## The hard part

Copying a slide from one PPTX to another isn't just deepcopying the slide's XML. A slide references:
- **Layout** (which references master)
- **Image parts** (background PNGs, embedded media)
- **Theme** (colors, fonts referenced indirectly)
- **Relationships (`rels`)** connecting all the above

When you cross PPTX files, all these relationships break. The recipes here handle each piece.

**Path G's output simplifies this.** Pages generated by Path G have a known structure:
- One full-slide PICTURE shape (the rasterized HTML background)
- Several TEXT_BOX shapes overlaid for editable text
- No complex layouts, master inheritance, or theme references

This makes the splice reliable. **G' should only be used to insert pages produced by Path G** — splicing arbitrary slides from arbitrary PPTs is much harder and out of scope.

## Recipe 1: Append new pages to end of original deck

The simplest case. Use this first to validate the approach works.

```python
from pptx import Presentation
from copy import deepcopy


def append_slides(original_path, new_pages_path, output_path):
    """Append all slides from new_pages_path to the end of original_path."""
    
    original = Presentation(original_path)
    new_deck = Presentation(new_pages_path)
    
    for new_slide in new_deck.slides:
        copy_slide_into_deck(new_slide, original)
    
    original.save(output_path)


def copy_slide_into_deck(source_slide, target_prs):
    """Copy a slide (with all its shapes and images) into target presentation.
    
    Creates a new blank slide at the end of target, then copies source's shapes XML.
    Image parts are cloned across PPTXs as needed.
    """
    # 1. Create a blank slide using a layout that exists in target
    #    We use the target's first blank-ish layout, since we're replacing all content anyway
    blank_layout = pick_blank_layout(target_prs)
    new_slide = target_prs.slides.add_slide(blank_layout)
    
    # 2. Remove default placeholders that come with the layout
    for shape in list(new_slide.shapes):
        if shape.is_placeholder:
            sp = shape._element
            sp.getparent().remove(sp)
    
    # 3. For each shape on source, clone it onto new_slide
    for source_shape in source_slide.shapes:
        clone_shape_to_slide(source_shape, new_slide)
    
    return new_slide


def pick_blank_layout(prs):
    """Pick the blank-est available layout (preferring 'Blank', falling back to last)."""
    for layout in prs.slide_layouts:
        if layout.name in ("Blank", "空白"):
            return layout
    # Fallback: last layout (often blank or near-blank in default themes)
    return prs.slide_layouts[-1]
```

## Recipe 2: Clone a shape across PPTXs

This is the workhorse. Handles text frames trivially, but pictures require part cloning.

```python
from pptx.util import Emu
from pptx.enum.shapes import MSO_SHAPE_TYPE


def clone_shape_to_slide(source_shape, target_slide):
    """Clone a single shape from one slide to another (possibly different PPTX)."""
    
    shape_type = source_shape.shape_type
    
    if shape_type == MSO_SHAPE_TYPE.PICTURE:
        clone_picture(source_shape, target_slide)
    elif shape_type == MSO_SHAPE_TYPE.GROUP:
        clone_group(source_shape, target_slide)
    elif source_shape.has_text_frame:
        clone_text_shape(source_shape, target_slide)
    elif shape_type == MSO_SHAPE_TYPE.AUTO_SHAPE:
        clone_auto_shape(source_shape, target_slide)
    else:
        # Generic XML-copy fallback. Works for shapes whose entire definition
        # lives in the slide's spTree without external part refs.
        clone_generic_xml(source_shape, target_slide)


def clone_picture(source_shape, target_slide):
    """Clone a PICTURE shape, including its image part."""
    # Read the image bytes from source
    image = source_shape.image
    image_bytes = image.blob
    
    # Write to a temp file (add_picture needs a path or file-like object)
    import io
    image_stream = io.BytesIO(image_bytes)
    
    # Add picture to target slide at same position/size
    new_pic = target_slide.shapes.add_picture(
        image_stream,
        left=source_shape.left,
        top=source_shape.top,
        width=source_shape.width,
        height=source_shape.height,
    )
    
    # Preserve rotation if any
    if source_shape.rotation != 0:
        new_pic.rotation = source_shape.rotation
    
    return new_pic


def clone_text_shape(source_shape, target_slide):
    """Clone a TEXT_BOX or text-bearing AUTO_SHAPE."""
    # For text boxes, the simplest reliable approach is to add a new text box
    # and copy the text + per-run formatting.
    from pptx.util import Emu
    
    new_tb = target_slide.shapes.add_textbox(
        left=source_shape.left,
        top=source_shape.top,
        width=source_shape.width,
        height=source_shape.height,
    )
    
    # Copy text and per-run formatting
    new_tf = new_tb.text_frame
    new_tf.word_wrap = source_shape.text_frame.word_wrap
    
    # Remove the default empty paragraph that add_textbox creates
    for p in list(new_tf.paragraphs):
        p._p.getparent().remove(p._p)
    
    # Copy paragraphs and runs from source
    for src_p in source_shape.text_frame.paragraphs:
        new_p = new_tf.add_paragraph()
        if src_p._pPr is not None:
            new_p._p.insert(0, deepcopy(src_p._pPr))
        new_p.alignment = src_p.alignment
        
        for src_r in src_p.runs:
            new_r = new_p.add_run()
            new_r.text = src_r.text
            copy_run_format(src_r, new_r)
    
    return new_tb


def copy_run_format(src_run, dst_run):
    """Same helper as in e2-format-tweak.md."""
    sf, df = src_run.font, dst_run.font
    if sf.name is not None: df.name = sf.name
    if sf.size is not None: df.size = sf.size
    if sf.bold is not None: df.bold = sf.bold
    if sf.italic is not None: df.italic = sf.italic
    try:
        if sf.color.rgb is not None:
            df.color.rgb = sf.color.rgb
    except (AttributeError, ValueError):
        pass


def clone_auto_shape(source_shape, target_slide):
    """Clone an AUTO_SHAPE (rectangle, oval, etc.) preserving fill and geometry."""
    from pptx.util import Emu
    
    new_shape = target_slide.shapes.add_shape(
        autoshape_type_id=source_shape.auto_shape_type,
        left=source_shape.left,
        top=source_shape.top,
        width=source_shape.width,
        height=source_shape.height,
    )
    
    # Copy fill
    try:
        if source_shape.fill.type == 1:  # SOLID
            new_shape.fill.solid()
            new_shape.fill.fore_color.rgb = source_shape.fill.fore_color.rgb
    except (AttributeError, ValueError):
        pass
    
    # Copy line (border)
    try:
        if source_shape.line.color.rgb is not None:
            new_shape.line.color.rgb = source_shape.line.color.rgb
        else:
            new_shape.line.fill.background()  # no line
    except (AttributeError, ValueError):
        pass
    
    # Copy text if any
    if source_shape.has_text_frame and source_shape.text_frame.text.strip():
        # Re-use text-shape copy logic
        new_tf = new_shape.text_frame
        for p in list(new_tf.paragraphs):
            p._p.getparent().remove(p._p)
        for src_p in source_shape.text_frame.paragraphs:
            new_p = new_tf.add_paragraph()
            if src_p._pPr is not None:
                new_p._p.insert(0, deepcopy(src_p._pPr))
            for src_r in src_p.runs:
                new_r = new_p.add_run()
                new_r.text = src_r.text
                copy_run_format(src_r, new_r)
    
    return new_shape


def clone_group(source_shape, target_slide):
    """Recursively clone shapes in a group. Doesn't recreate the group itself
    (python-pptx has no high-level group API) — just clones the children to top level."""
    for child in source_shape.shapes:
        clone_shape_to_slide(child, target_slide)


def clone_generic_xml(source_shape, target_slide):
    """Last-resort: deepcopy the source shape's XML into the target slide.
    Only safe if the shape has no external part refs (no pictures, no embedded media)."""
    new_el = deepcopy(source_shape._element)
    target_slide.shapes._spTree.insert_element_before(new_el, 'p:extLst')
```

## Recipe 3: Insert at specific position (not append)

Building on Recipe 1, but with position control.

```python
def insert_slides_at_position(original_path, new_pages_path, output_path, position):
    """Insert new pages at `position` (1-based). Original pages shift down."""
    
    original = Presentation(original_path)
    new_deck = Presentation(new_pages_path)
    
    # Track which slides were just added (we'll need to move them)
    initial_count = len(original.slides)
    
    # Append all new slides first
    for new_slide in new_deck.slides:
        copy_slide_into_deck(new_slide, original)
    
    # Now move the newly appended slides into position
    num_new = len(new_deck.slides)
    slide_id_list = original.slides._sldIdLst
    all_ids = list(slide_id_list)
    
    new_ids = all_ids[initial_count:]  # the ones we just added
    
    # Remove them from end
    for sid in new_ids:
        slide_id_list.remove(sid)
    
    # Insert at target position (convert 1-based to 0-based)
    insert_idx = position - 1
    for offset, sid in enumerate(new_ids):
        slide_id_list.insert(insert_idx + offset, sid)
    
    original.save(output_path)
```

## Recipe 4: Replace a slide

```python
def replace_slide(original_path, new_pages_path, output_path, position):
    """Replace the slide at `position` (1-based) with the new pages.
    If new_pages has multiple slides, all of them are inserted at position."""
    
    original = Presentation(original_path)
    new_deck = Presentation(new_pages_path)
    
    initial_count = len(original.slides)
    
    # Step 1: Append all new slides
    for new_slide in new_deck.slides:
        copy_slide_into_deck(new_slide, original)
    
    # Step 2: Remove the target slide
    slide_id_list = original.slides._sldIdLst
    target_idx = position - 1  # 1-based to 0-based
    all_ids = list(slide_id_list)
    
    # The new slides are at the end (indexes initial_count..)
    new_ids = all_ids[initial_count:]
    
    # Remove new slides from end so they're not in the way of deletion
    for sid in new_ids:
        slide_id_list.remove(sid)
    
    # Now delete the target slide
    target_sid = list(slide_id_list)[target_idx]
    rId = target_sid.attrib['{http://schemas.openxmlformats.org/officeDocument/2006/relationships}id']
    original.part.drop_rel(rId)
    slide_id_list.remove(target_sid)
    
    # Step 3: Insert the new slides at target position
    for offset, sid in enumerate(new_ids):
        slide_id_list.insert(target_idx + offset, sid)
    
    original.save(output_path)
```

## Edge cases and gotchas

| Issue | Symptom | Handling |
|---|---|---|
| Source slide has shapes with theme-color refs | Color comes out wrong in target | Path G output uses explicit RGB, so this shouldn't happen for G' inputs. If it does, escalate. |
| Image part rel lost | Inserted page renders white/blank | Recipe 2's `clone_picture` re-creates the part via `add_picture`. If this fails, the image bytes were unreadable — usually means source PPT was corrupted. |
| Slide layout naming differs across PPTXs | `pick_blank_layout` may return a non-blank layout | The recipe removes default placeholders after slide creation, so even a non-blank layout works. |
| Original deck uses CJK fonts not present in environment | New pages may render in wrong fallback font when opened on a different machine | Same as Path G — warn user about font portability. |
| Two slides have shapes with conflicting IDs | python-pptx generates new IDs on add, so this is handled | No action needed. |

## Verification after splice

After running any of the recipes:

```python
result = Presentation(output_path)
print(f"Final slide count: {len(result.slides)}")

# Spot-check the inserted slides
for idx, slide in enumerate(result.slides, 1):
    n_shapes = len(slide.shapes)
    n_pics = sum(1 for s in slide.shapes if s.shape_type == MSO_SHAPE_TYPE.PICTURE)
    n_texts = sum(1 for s in slide.shapes if s.has_text_frame)
    print(f"  Slide {idx}: {n_shapes} shapes ({n_pics} pictures, {n_texts} text)")
```

If an inserted slide shows 0 shapes, the copy failed. Diagnose via `pptx_inspect.py` on both source and output.
